📊 Social Media Analysis Project (SQL)
📌 Project Overview

This project focuses on analyzing user behavior and engagement patterns on a social media platform using SQL. The goal is to extract meaningful insights from user interactions such as posts, likes, comments, follows, and hashtags to support data-driven decision-making.

🗂️ Database Schema

The project is based on a relational database ig_clone consisting of the following tables:

Users – Stores user profile details and account creation time

Photos – Contains user posts with timestamps

Likes – Records likes on posts

Comments – Stores user comments on posts

Follows – Represents follower-following relationships

Tags – Stores hashtags

Photo_Tags – Maps hashtags to posts

🎯 Objectives

Identify top engaged users based on likes and comments

Analyze user activity levels and detect inactive users

Evaluate engagement trends across hashtags/content topics

Understand posting patterns and time-based engagement

Segment users for targeted marketing strategies

Identify potential influencers and brand advocates

📈 Key Metrics

Total Posts per User

Likes & Comments Received

Engagement Rate per Post

Follower Count

Hashtag-wise Engagement

Active vs Inactive Users

📊 Analysis & Insights

High-engagement users contribute significantly to platform activity and can be leveraged as influencers

Certain hashtags consistently generate higher engagement, indicating strong content preferences

A portion of users remain inactive, representing an opportunity for re-engagement strategies

Engagement varies across posting times, highlighting optimal time windows for content publishing

💡 Recommendations

Promote high-performing content topics and hashtags

Collaborate with top engaged users for influencer campaigns

Implement re-engagement strategies for inactive users

Optimize posting schedules based on peak engagement hours

Use insights to design targeted and personalized ad campaigns

🛠️ Tools & Technologies

SQL (MySQL)

Excel (for visualization)

GitHub (project hosting)

📌 Conclusion

This project demonstrates how SQL can be effectively used to analyze social media data, uncover user behavior patterns, and generate actionable insights to improve engagement, retention, and marketing strategies.
